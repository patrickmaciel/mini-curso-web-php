Elementos com atributos flex: display: flex;

No exemplo abaixo com o uso do display: flex; os elementos agora ficaram em linha (inline/row) ao invés de em bloco (block/column).

Usando somente essa configuração os elementos forçam o máximo possível para se encaixarem dentro do container principal (pai), e se não der, eles ultrapassam o limite e acabam saindo pra fora.
